# TouchSpace Academy: связанная структура

Что изменено:

- Добавлен верхнеуровневый раздел `sections.html` — карта Академии.
- Левое меню теперь ведёт в: Главная → Разделы → Курсы → Бренды → Задачи → Избранное → Кабинет → Создание тестов.
- Курсы стали контейнерами: `course-*.html`.
- Уроки находятся внутри курсов: `lesson-*.html`, плюс существующие `pergo-brand.html` и `calitex-brand.html` используются как брендовые уроки.
- Итоговые тесты привязаны к курсам: `pergo-test.html`, `calitex-test.html` и статические заготовки `lesson-*-test.html`.
- Карточки курсов в `courses.html` кликабельны и ведут на страницы курсов.
- Добавлены заготовки разделов: библиотека, глоссарий, сообщества, опросы, рейтинг, мероприятия, отчёты, банк идей, квесты, справочник сотрудника.

Запуск:

```bash
python3 -m http.server 8000
```

Открыть: `http://localhost:8000/`.
