# TouchSpace Академия

Локальная статическая версия обучающей платформы.

## Как открыть

1. Распакуйте архив.
2. Откройте файл `index.html` в браузере.

## Страницы

- `index.html` — главная страница.
- `courses.html` — каталог курсов.
- `brands.html` — каталог брендов, категорий и коллекций.
- `pergo-brand.html` — шаблонная страница бренда Pergo.
- `calitex-brand.html` — шаблонная страница бренда Calitex.
- `tasks.html` — задачи и календарь обучения.

## Важно

Не удаляйте папку `icons`: в ней лежат иконки и изображения для страниц.

Общая шапка подключается файлами `shared-header.css` и `shared-header.js`. Для новых страниц достаточно добавить обычный `<header class="header">` с нужным поиском или пустой `<header class="header" data-search-placeholder="..."></header>` — правый блок пользователя и уведомлений будет собран автоматически.
