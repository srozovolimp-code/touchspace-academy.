# Advanced Pattern

Эта ветка содержит жесткий шаблон для будущих разделов формата Advanced.

За основу взята структура `alpine-floor-advanced`: академия бренда, семь учебных блоков, страницы внутри блоков, промежуточные тесты и итоговый тест. Брендовый и учебный текст очищен и заменен на шаблонные поля вида `{{BRAND_NAME}}`, `{{MODULE_01_TITLE}}`, `{{QUESTION_TEXT}}`.

## Основные файлы

- `advanced-pattern-academy.html` — входная страница академии.
- `advanced-pattern-course-manifest.js` — карта модулей, ссылок, времени и уровней.
- `advanced-pattern-block-01.html` ... `advanced-pattern-block-07.html` — страницы модулей.
- `advanced-pattern-block-01-data.js` ... `advanced-pattern-block-07-data.js` — учебный контент и тесты модулей.
- `advanced-pattern-final-test.html` и `advanced-pattern-final-test-data.js` — итоговая проверка.

## Правила использования

1. Скопировать набор `advanced-pattern-*` в новые файлы бренда.
2. Заменить `advanced-pattern` на slug нового раздела.
3. Заменить `AdvancedPattern` на PascalCase-имя нового раздела.
4. Заполнить все поля `{{...}}` контентом бренда.
5. Добавить новый раздел в `brand-academy.js` и карточку в каталог брендов, если раздел должен быть виден на сайте.

Технические идентификаторы, типы блоков, типы вопросов и связи тестов оставлены рабочими. Их можно переименовывать, но нельзя ломать соответствие между `связанный_тест`, `связанная_страница`, `url`, `data_file` и глобальными именами данных.
